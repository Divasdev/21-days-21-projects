Essential JS Concepts for React

A quick revision guide for Project 19-20

ES6

1. Arrow Functions

A shorter syntax for writing function expressions. They don't have their own 'this' binding.

// Traditional
function greet(name) {
  return "Hello " + name;
}

// Arrow Function
const greet = (name) => `Hello ${name}`;
ES6

2. Destructuring

Extract values from arrays or properties from objects into distinct variables.

// Object
const user = { name: 'Alex', age: 24 };
const { name, age } = user;

// Array
const colors = ['red', 'green'];
const [primary, secondary] = colors;
Array Methods

3. Map()

Creates a new array by applying a function to every element in an existing array. Crucial for rendering lists in React.

const nums = [1, 2, 3];
const doubled = nums.map((num) => num * 2);
// [2, 4, 6]
Array Methods

4. Filter()

Creates a new array with all elements that pass the test implemented by the provided function.

const ages = [15, 22, 10, 30];
const adults = ages.filter((age) => age >= 18);
// [22, 30]
ES6

5. Spread Operator (...)

Expands an iterable (like an array) into more elements. Useful for copying arrays/objects or combining them. 

const oldArr = [1, 2];
const newArr = [...oldArr, 3, 4];
// [1, 2, 3, 4]

const user = { name: 'John' };
const updatedUser = { ...user, age: 30 };
ES6

6. Modules (Import/Export)

Breaking code into separate files.

// file1.js
export const name = "React";

// file2.js
import { name } from './file1.js';
